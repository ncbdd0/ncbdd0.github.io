2.纸牌连连看

纸牌：总数54张纸牌，含大王、小王。

发牌：首轮按顺序每人发7张牌。

出牌规则：对子、三张以上的顺子、三带一可以出牌。

补牌规则：出几张牌就可以补几张牌。

结束规则，以下任意条件之一被触发：

1.所有纸牌都被抽完。
2.所有玩家手上的牌全部出完。
3.所有玩家手上的牌都是7 张，且无法出牌。
4.某个玩家手上的牌全部出完。

单局胜者规则（只有一个胜者）：

1.所有纸牌都被抽完、所有玩家手上的牌全部出完：已经出牌的牌面总和最高者获得胜利。
2.某个玩家手上的牌全部出完：该玩家获胜。

积分制胜者： >1.按局积分，胜1局积1分。
>2.积分高者获得最终胜利。

def straight(num_list):




    #过滤大小王


    valid = sorted(set(n for n in num_list if n != 0))


    if len(valid) < 4:


        return None


    max = []


    current = [valid[0]]


    for n in valid[1:]:


        if n == current[-1] + 1:


            current.append(n)


            if len(current) >= 4:


                max = current.copy() if not max or len(current) > len(max) else max


        else:


            current = [n]


    return max

def dict_query_key(DCT,str):


    for char in DCT.keys():


        if char == str:


            return DCT[char]


    return "未找到"




def dict_query_value(DCT,str):


    for key,value in DCT.items():


        if value == str:


            return key


    return "未找到"

使用组合数据类型实现一个学生信息管理系统

要求：实现一个学生信息管理系统，使用字典和列表等组合数据类型存储学生信息和成绩，并实现以下功能：

  1. 添加学生信息：包括姓名、年龄、性别、学号和成绩等；

  2. 查询学生信息：根据学生姓名或学号查询学生信息；

  3. 修改学生信息：根据学生姓名或学号修改学生信息；

  4. 删除学生信息：根据学生姓名或学号删除学生信息；

  5. 显示所有学生信息：按照学生学号排序，列出所有学生的信息；

  6. 统计学生成绩：给出平均分最高的前3名学生信息，给出各科成绩的平均分和中位数。

要求：以上1-6的功能都用函数来实现。

数据基本结构如下：


{学号1：{姓名：XXX，年龄：XXX，性别：XXX，成绩：[XX,XX,XX]}，


学号2：{姓名：XXX，年龄：XXX，性别：XXX，成绩：[XX,XX,XX]}，


学号3：{姓名：XXX，年龄：XXX，性别：XXX，成绩：[XX,XX,XX]}，


学号4：{姓名：XXX，年龄：XXX，性别：XXX，成绩：[XX,XX,XX]},


学号5：{姓名：XXX，年龄：XXX，性别：XXX，成绩：[XX,XX,XX]} ]


其中：成绩的存储按顺序为 [高数成绩，线代成绩，英语成绩]


如：


3501214:{姓名：张三，年龄：20，性别：女，成绩：[98,77,65]}
