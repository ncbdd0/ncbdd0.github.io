volatility2.6常用命令

1.判断镜像系统版本信息

vol.py -f mem.vmem imageinfo

知道系统后后面的命令前面加上-–profile=[系统]。

2.列举进程

vol.py -f mem.vmem –profile=[系统] pslist

3.可识别子进程以及父进程

vol.py -f mem.vmem –profile=[系统] pstree

4.可显示被隐藏的进程

vol.py -f mem.vmem –profile=[系统] psscan

5.查看注册表

vol.py -f mem.vmem –profile=[系统] hivelist

6.查看sid

vol.py -f mem.vmem –profile=[系统] getsids -p [PID号]

7. 提取进程

vol.py -f mem.vmem –profile=[系统] memdump -p [PID号] -D [dump出的文件保存的目录]

也可以换成把-p [PID号]换成–offset [地址]。

8.导出注册表

vol.py -f mem.vmem –profile=[系统] hivedump -o [注册表的 virtual 地址]

9.列出SAM表中的用户

vol.py -f mem.vmem –profile=[系统] printkey -K “SAM\Domains\Account\Users\Names”

10.扫描电脑中的文件

vol.py -f mem.vmem –profile=[系统] filescan

11.导出列举的文件

vol.py -f mem.vmem –profile=[系统] dumpfiles -Q [地址] –D [路径]

12.提取cmd命令使用情况

vol.py -f mem.vmem –profile=[系统] cmdscan

13.获取到当时的网络连接情况

vol.py -f mem.vmem –profile=[系统] netscan

14.获取IE浏览器的使用情况

vol.py -f mem.vmem –profile=[系统] iehistory

15.获取用户密码

vol.py -f mem.vmem –profile=[系统] mimikatz（需要安装插件）

16.查看剪切板内容

vol.py -f mem.vmem –profile=[系统] clipboard
