python传输文件

2025/11/11 学习 python 文件

参考：python互传文件

1.启动http服务器

在你想要共享文件的那台机器上，打开终端并导航到包含文件的目录。

cd file_path

然后启动http服务器：

python -m http.server 8000

然后会启动一个HTTP服务器，监听8000端口，可以选择其他端口，只需替换端口号即可。

2.获取ip地址

使用命令获取ip地址：（类Unix系统）

ifconfig

Windows：

ipconfig

3.访问共享文件

在网络中的其他设备（比如你的手机、另一个电脑等）上打开浏览器，访问以下 URL：

http://your_ip_address:8000

会看到一个简单的文件目录列表，点击列表中的文件链接即可下载文件。
