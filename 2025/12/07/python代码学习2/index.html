1.两数之和

给定一个整数数组nums和一个整数目标值target，请你在该数组中找出和为目标值target的那两个整数，并返回它们的数组下标。

示例 1：

输入：nums = [2,7,11,15], target = 9

输出：[0,1]

示例 2：

输入：nums = [3,2,4], target = 6

输出：[1,2]

示例 3：

输入：nums = [3,3], target = 6

输出：[0,1]

刚开始还不知道不用自己编写主函数给我一顿爆红……

代码:

class Solution(object):
    def twoSum(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: List[int]
        """
        for i in range(len(nums)):
            res = target - nums[i]
            if res in nums[i+1:]:
                return i ,nums[i+1:].index(res)+i+1

思路：通过计算看下一个要找的数字是否存在然后利用index()返回索引。

2.罗马数字转整数

罗马数字包含以下七种字符: I， V， X， L，C，D 和 M。

字符 数值
I 1
V 5
X 10
L 50
C 100
D 500
M 1000

例如，罗马数字2写做II，即为两个并列的1。12写做XII，即为X+II。27写做XXVII, 即为XX+V+II 。

通常情况下，罗马数字中小的数字在大的数字的右边。但也存在特例，例如4不写做IIII，而是IV。数字1在数字 5的左边，所表示的数等于大数5减小数1得到的数值4。同样地，数字9表示为IX。这个特殊的规则只适用于以下六种情况：

  • I 可以放在 V (5) 和 X (10) 的左边，来表示 4 和 9。
  • X 可以放在 L (50) 和 C (100) 的左边，来表示 40 和 90。
  • C 可以放在 D (500) 和 M (1000) 的左边，来表示 400 和 900。

给定一个罗马数字，将其转换成整数。

示例 1:

输入: s = “III”

输出: 3

示例 2:

输入: s = “IV”

输出: 4

示例 3:

输入: s = “MCMXCIV”

输出: 1994

解释: M = 1000, CM = 900, XC = 90, IV = 4.

代码：
1.初版菜鸟版

class Solution(object):

        """
        :type s: str
        :rtype: int
        """
        nums = list(s)
        sum,j = 0,0
        while j < len(nums): 
            if nums[j] == 'I':
                if j + 1 < len(nums) and nums[j+1] == 'V':
                    sum += 4
                    j += 1
                elif j + 1 < len(nums) and nums[j+1] == 'X':
                    sum += 9
                    j += 1
                else:
                    sum += 1
            elif nums[j] == 'V':
                sum += 5
            elif nums[j] == 'X':
                if j + 1 < len(nums) and nums[j+1] == 'L':
                    sum += 40
                    j += 1
                elif j + 1 < len(nums) and nums[j+1] == 'C':
                    sum += 90
                    j += 1
                else:
                    sum += 10
            elif nums[j] == 'L':
                sum += 50
            elif nums[j] == 'C':
                if j + 1 < len(nums) and nums[j+1] == 'D':
                    sum += 400
                    j += 1
                elif j + 1 < len(nums) and nums[j+1] == 'M':
                    sum += 900
                    j += 1
                else:
                    sum += 100
            elif nums[j] == 'D':
                sum += 500
            elif nums[j] == 'M':
                sum += 1000
            else:
                continue
            j += 1
        return sum

这个做的真没什么逻辑，就是根据题目描述简单地用了while循环和if分支，敲了几十行我自己看着都头大。

2.优化版

class Solution(object):
    def romanToInt(self, s):
        """
        :type s: str
        :rtype: int
        """
        nums = list(s)
        hashmap = {"I" : 1 , "V" : 5 , "X" : 10 , "L" : 50 , "C" : 100 , "D" : 500 , "M": 1000}
        sum , i = 0 , 0
        while i < len(nums):
            if i+1 < len(nums) and nums[i]+nums[i+1] in ['IV','IX','XL','XC','CD','CM']:
                sum += hashmap[nums[i+1]] - hashmap[nums[i]]
                i += 1
            else:
                sum += hashmap[nums[i]]
            i += 1
        return sum

在看了一些佬的解析后引入了字典，虽然就提速了1ms(bushi，但是更好理解代码量也相对少点hhh。

思路：遍历字符串后通过哈希图根据键查找对应的值，遇到特殊规则的时候注意再多跳一格。

3.最长公共前缀

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 “”。

示例 1：

输入：strs = [“flower”,”flow”,”flight”]

输出：”fl”

示例 2：

输入：strs = [“dog”,”racecar”,”car”]

输出：””

代码：

class Solution(object):
    def longestCommonPrefix(self, strs):
        """
        :type strs: List[str]
        :rtype: str
        """
        i , k= 1 , 0
        same = []
        num = list(strs[0])
        if len(strs) == 1:
            return strs[0]
        while i < len(strs):
            current_k = 0
            num_str = list(strs[i])
            if len(num_str) > len(num):
                length = len(num)
            else:
                length = len(num_str)
            for j in range(length):
                if num_str[j] == num[j]:
                    current_k = j+1
                else:
                    break
            if i == 1:
                k = current_k
            else:
                k = min(current_k,k)
            i += 1
        if k == 0:
            return ""
        else:
            for i in range(k):
                same.append(num[i])
            result = ''.join(same)
            return result

脑子总是在犯蠢，做的时候越界问题卡了我一会orz。

思路：把后面的字符串跟第一个字符串进行对比，然后不断更新最小的公共前缀数，还要注意排除一下只有一个数组中字符串的情况。

4.有效的括号

给定一个只包括 ‘(‘，’)’，’{‘，’}’，’[‘，’]’ 的字符串 s ，判断字符串是否有效。

有效字符串需满足：

  • 左括号必须用相同类型的右括号闭合。
  • 左括号必须以正确的顺序闭合。
  • 每个右括号都有一个对应的相同类型的左括号。

示例 1：

输入：s = “()[]{}”

输出：true

示例 2：

输入：s = “([])”

输出：true

示例 3：

输入：s = “([)]”

输出：false

代码：

class Solution(object):
    def isValid(self, s):
        """
        :type s: str
        :rtype: bool
        """
        strs = list(s)
        stack = []
        pairs = {')' : '(' , ']' : '[' , '}' : '{'}
        for i in range(len(strs)):
            if strs[i] in pairs.values():
                stack.append(strs[i])
            elif strs[i] in pairs.keys():
                if not stack or stack[-1] != pairs[strs[i]]:
                    return False
                else:
                    stack.pop()
            else:
                return False
        return not stack

这题按正确的顺序闭合那块没做出来最后问ai了orz，用的是栈的方法。

思路：先把左括号压入栈，然后查找到的右括号看是否与栈顶匹配（从内到外的第一个左括号即要求正确的顺序闭合规则），如匹配则把对应的左括号弹出，最后栈空则为有效括号字符串。

5.合并两个有序链表

将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

示例 1：

输入：l1 = [1,2,4], l2 = [1,3,4]

输出：[1,1,2,3,4,4]

示例 2：

输入：l1 = [], l2 = [0]

输出：[0]

代码：

# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution(object):
    def mergeTwoLists(self, list1, list2):
        """
        :type list1: Optional[ListNode]
        :type list2: Optional[ListNode]
        :rtype: Optional[ListNode]
        """
        dummy = ListNode(0)#创建头结点
        current = dummy
        while list1 and list2:
            if list1.val <= list2.val:
                current.next = list1
                list1 = list1.next
            else:
                current.next = list2
                list2 = list2.next
            current = current.next
        current.next = list1 if list1 else list2
        return dummy.next

这题我刚开始做的时候直接当列表做的，后面问了ai才发现list1和list2是链表节点不能直接当成列表orz。

思路：先创建一个头结点然后将list1和list2指向的元素进行比较，按升序连接指针后返回。

这里学到了一个三目运算符知识点：x = a if condition else b

等价于：

if condition:
    x = a
else:
    x = b
6.删除有序数组中的重复项

给你一个 非严格递增排列的数组nums，请你原地删除重复出现的元素，使每个元素只出现一次，返回删除后数组的新长度。元素的相对顺序应该保持一致。然后返回nums中唯一元素的个数。

考虑 nums 的唯一元素的数量为k。去重后，返回唯一元素的数量k。

nums的前k个元素应包含排序后的唯一数字。下标k-1之后的剩余元素可以忽略。

示例 1：

输入：nums = [0,0,1,1,1,2,2,3,3,4]

输出：5, nums = [0,1,2,3,4,,,,,_]

解释：函数应该返回新的长度 5 ， 并且原数组 nums 的前五个元素被修改为 0, 1, 2, 3, 4 。不需要考虑数组中超出新长度后面的元素。

代码：

class Solution(object):
    def removeDuplicates(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        k = 0
        for i in range(1, len(nums)):
            if nums[i] != nums[k]:
                k += 1
                nums[k] = nums[i]
        return k + 1

一个简单的双指针应用，平时双指针用的比较少就记录一下。

思路：设立一个慢指针k和快指针i，k指向最后一个不重复元素的位置，当i找到新的不重复元素后让k后移。

7.找出字符串中第一个匹配项的下标

给你两个字符串haystack和needle，请你在haystack字符串中找出needle字符串的第一个匹配项的下标（下标从0开始。如果needle不是haystack的一部分，则返回-1 。

示例 1：

输入：haystack = “sadbutsad”, needle = “sad”

输出：0

示例 2：

输入：haystack = “leetcode”, needle = “leeto”

输出：-1

代码：

自己做的菜鸡版：

class Solution(object):
    def strStr(self, haystack, needle):
        """
        :type haystack: str
        :type needle: str
        :rtype: int
        """
        if len(needle) > len(haystack):
            return -1
        elif len(needle) == len(haystack):
            if needle == haystack:
                return 0
            else:
                return -1
        else:
            if len(needle) == 1:
                for i in range(len(haystack)):
                    if needle == haystack[i]:
                        return i
                return -1
            else:
                for i in range(len(haystack)-len(needle)+1):
                    if haystack[i:i+len(needle)] == needle:
                        return i
                return -1

我这个比较简单粗暴，就是直接把haystack中查找是否存在needle连续的字符串然后排除一些特殊情况。

师傅的简单版：

class Solution(object):
    def strStr(self, haystack, needle):
        """
        :type haystack: str
        :type needle: str
        :rtype: int
        """
        nums = []
        index = 0 
        for i in haystack.split(needle):
            nums.append(i)
        if len(nums) == 1:
            return -1
        elif nums[0] == "":
            return 0
        else:
            return len(nums[0])

思路：通过split()在haystack中去除needle字符串，如果取出后的数组长度为1则说明不存在（因为不存在split会返回一个原字符串），如果第一个字符为空则在开头，若在中间存在则下标就是被去除后的长度。
