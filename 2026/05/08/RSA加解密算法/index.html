RSA加解密算法

2026/5/8 算法 密码学

这个算法比较简单，就不在网上找别的师傅的文章了，直接拿老师的ppt随便写写：）

RSA算法整体结构

RSA算法包括密钥生成算法、加密算法和解密算法。

RSA算法中的一对密钥分为加密密钥（即公钥）PK和解密密钥（即私钥）SK，公钥PK是公开信息，私钥SK需要保密。

加密时使用公钥PK对明文进行加密，得到密文，解密时使用私钥SK对密文进行解密，得到明文。

密钥生成算法

  1. 任选两个不同的大素数 p 和 q ，计算n = p * q，φ(n) = (p−1) * (q−1)。
  2. 任选一个大整数e，满足gcd⁡(e , φ(n)) = 1且1 < e < n，e为公钥PK。
  3. 作为私钥SK的d，应满足d * e mod φ(n) = 1，即d * e = k * φ(n) + 1，其中，k ≥ 1且是整数。
  4. 公开整数n和e，秘密保存d。

加密算法

将明文m（m<n，是一个整数）加密成密文c，加密算法为：c = E(m) = m^e mod n

解密算法

将密文c解密为明文m，解密算法为：m = D(c) = c^d mod n

一个仍需完善的代码实现，我没用random库，是手动输入p和q还有e的，这样可以输入老师ppt里的例子orz。

import math


import time




# 判断互质


def judge_gcd(a , b):


    return math.gcd(abs(a) , abs(b)) == 1




# 判断素数


def judge_prime(n):


    if n < 2 or (n % 2 == 0 and n != 2):


        return False


    if n == 2:


        return True


    # 遍历到根号n


    for i in range(3, int(math.sqrt(n)) + 1, 2):


        if n % i == 0:


            return False


    return True




# 扩展欧几里的算法 计算ax + by = 1 中的x与y的整数解（a与b互质）, x为逆元


def ext_gcd(a , b):


    if b == 0:


        x1 , y1 = 1 , 0


        x , y = x1 , y1


        r = a


        return r , x , y


    else:


        r , x1 , y1 = ext_gcd(b , a % b)


        x = y1


        y = x1 - a // b * y1


        return r , x , y




# 超大整数超大次幂对超大的整数取模


def exp_mode(base , exponent , n):


    bin_array = bin(exponent)[2:][::-1]


    r = len(bin_array)


    base_array = []




    pre_base = base


    base_array.append(pre_base)




    for _ in range(r - 1):


        next_base = (pre_base * pre_base) % n


        base_array.append(next_base)


        pre_base = next_base


    a_w_b = _multi(base_array , bin_array , n)


    return a_w_b % n


    


# 大模运算辅助函数


def _multi(array , bin_array , n):


    result = 1


    for index in range(len(array)):


        a = array[index]


        if not int(bin_array[index]):


            continue


        result *= a


        result = result % n


    return result




# 加密算法


def crypt(m , e , n):


    return exp_mode(m , e , n)




# 解密算法


def decrypt(c , d , n):


    return exp_mode(c , d , n)




# 主函数


def main():


    while True:


        p , q = map(int , input("请输入两个不同的素数p , q(空格隔开)：").strip().split())


        if p != q and judge_prime(p) and judge_prime(q):


            break


        print("输入错误请重试")


    n = p * q # 公钥 n


    n_4 = (p - 1) * (q - 1) # 计算欧拉函数


    while True:


        e = int(input("请输入公钥e："))


        if judge_gcd(n_4 , e):


            break


        print("输入错误，e不与欧拉函数输入n的结果互质")


    r , x , y = ext_gcd(e , n_4) # d为私钥


    d = x % n_4


    print("=" * 5 , end = '')


    print("RSA算法" , end = '')


    print("=" * 5)


    while True:


        choose = int(input("加密选择1，解密选择2，退出程序选择0：").strip())


        if choose == 1:


            while True:


                m = int(input("请输入明文：").strip())


                if m >= n:


                    print(f"输入错误，m应小于{n}")


                    continue


                break


            c = crypt(m , e , n) # 密文


            print(f"密文为：{c}")


        elif choose == 2:


            c_k = int(input("请输入密文：").strip())


            m_k = decrypt(c_k , d , n) # 密文


            print(f"明文为：{m_k}")


        elif choose == 0:


            print("bye")


            break


        else:


            print("输入错误请重试")


    


main()
