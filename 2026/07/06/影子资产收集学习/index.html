影子资产收集学习

2026/7/6 攻防 信息搜集

把之前学习的笔记投上来存个档：）

至于完善，那就又是一个有生之年系列……

之前学习的参考文章（滑跪）：信息收集道道之外网信息收集

第一步：开源情报搜集

  • 知道靶标单位后，在ICP/IP地址/域名信息备案管理系统 查询该单位ICP备案信息
  • 在企查查搜索该单位，获取官网、邮箱等，重点在知识产权，其中包含
    • 小程序
    • APP
    • 公众号
    • 旗下网站 还有与其有关联的单位的信息也可以一并查询
  • 接着对于搜集到的主域名就可以开始进行子域名的搜索整合，然后去重
    • crt.sh：查询与主域名相关证书，提取出子域名的信息
    • fofa类测绘平台：如domain=“”
    • 常规搜索引擎：site:mi.com
    • 工具：subfinder等
    • github：从官网邮箱信息获得邮箱格式， 使用github高级语法搜索邮箱格式、单位名称等是否有泄露等
  • 对搜集到的子域名：
    • whois查询域名注册信息
    • ip138等查询历史记录

第二步：主动搜集

  • dnsx解析子域名获得ip
  • fofa搜索资产
  • 扫描IP C段获取旁站
  • 开始探测主机存活状态
  • 对存活的主机进行端口扫描，获取开放服务，指纹信息
  • spray等进行目录爆破
  • 对小程序进行反编译分析api接口和参数
  • katana等爬取站点js文件分析获取接口信息，同时可以搜索敏感参数

几个网站的基础语法是差不多的，就是换个格式：

功能 语法格式
标题中包含关键词的页面 title = “xxx”
页面内容中包含指定关键词的页面 body = “xxx”
协议类型 protocol = “http”
指定IP段 ip = ”x.x.x.x
端口 port = “8080”
域名 domain = “example.com”
域名 host = “xxx.com”
国家/地区 country=“CN”/region=“Fujian”
ASN编号 asn=“ASxxxx”
